WebGL Portfolio Experiment

Posted: 05/03/12
Experimenting with Mr. Doob's Three.js

Categories: Design Development Web

Tags: Development WebGL

This is an experiment/proof-of-concept of a WebGL exposition for my portfolio. The trees were modeled in Blender3D. The content is pulled in from a WordPress account via the JSON API, then rendered into a canvas object which is then copied as an image to be used as the source for a texture plane using three.js. To view an article simply click on it or use the arrow keys to move through them sequentially. The escape key will throw the camera back into the original path it had when the page loaded, orbiting the main tree.
